# Expense Tracker Application

## 🚀 Features
- Add, delete, and view expenses with description and amount
- Real-time calculation of total expenses
- Persistent data storage using browser localStorage
- Clean and responsive user interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Project Overview
This Expense Tracker is a frontend web application that helps users manage daily expenses efficiently. Users can record expenses, view the updated total instantly, and retain data across sessions using localStorage.

## 🧠 Implementation Details
- Used JavaScript event handling to manage user interactions such as adding and deleting expenses
- Stored expense data in browser localStorage to ensure persistence across page reloads
- Calculated total expenses dynamically using array operations
- Updated the DOM in real time to reflect data changes without refreshing the page
